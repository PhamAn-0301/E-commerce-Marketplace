# Multi-Vendor E-commerce Marketplace

## Cấu trúc thư mục mẫu

- `views/`: Giao diện Handlebars
- `routes/`: Định tuyến
- `middlewares/`: Middleware (auth, error, ...)
- `controllers/`: Xử lý logic
- `models/`: Định nghĩa schema
- `services/`: Xử lý nghiệp vụ
- `utils/`: Hàm tiện ích
- `public/`: Static files
- `config/`: Cấu hình
- `logs/`: Log hệ thống
- `test/`: Unit test
- `docs/`: Tài liệu
- `.env`: Biến môi trường
- `app.js`: Khởi tạo app
- `service.js`: Service chạy nền
